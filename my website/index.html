—— 学习路线 ——
learning path

HTML

HTML不是一种编程语言，而是一种标记语言，超文本标记语言，用于创建网页的标准标记语言， HTML定义了网页的结构，可以比作为“人的骨架、器官”

JavaScript

JavaScript是一种脚本语言，一种轻量级的编程语言。是可插入HTML页面的编程代码， JavaScript定义了网页中用户的操作行为，可比作为“人的活动动作”

CSS

CSS是层叠样式表，通过css可以分别为网页的每一块来设置样式， CSS定义了网页中的样式布局，可比作为“人的样貌外观”
